## вҡҷпёҸ Instrucciones de ejecuciГіn

### рҹҹў VersiГіn Cliente (JavaScript puro)

1. Ir a la carpeta `cliente`
2. Abrir el archivo `index.html` en el navegador

---

### рҹ”ө VersiГіn Cliente-Servidor con Node.js

1. Abrir una terminal en la carpeta `node`
2. Instalar dependencias:

   ```
   npm install
   ```

3. Ejecutar el servidor:

   ```
   node server.js
   ```

4. Abrir en el navegador:

   ```
   http://localhost:3000
   ```

---

### рҹҹЎ VersiГіn Cliente-Servidor con Python (Flask)

1. Abrir una terminal en la carpeta `python`
2. Instalar Flask:

   ```
   pip install flask
   ```

3. Ejecutar la aplicaciГіn:

   ```
   python app.py
   ```

4. Abrir en el navegador:

   ```
   http://127.0.0.1:5000
   ```
